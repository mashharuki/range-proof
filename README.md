# range-proof
BulletProofを使った範囲証明用のアプリを開発するためのリポジトリです。

## ビルドコマンド

```bash
cd rangeProof && cargo build
```

## プログラム実行コマンド

```bash
cd rangeProof && cargo run rangeProof
```
